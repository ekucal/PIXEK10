The pixek.py is a software package for PIXE experimental data processing based on filtering by Savitzky-Golay methods and background removal using Wavelets-based algorithms. It also serves for data analysis using FFT/CWT methods. This program has been written under Python3.x language, is under the testing phase in Windows/Linux environments. The code is free to download, distribute, and modify under GNU license conditions.
